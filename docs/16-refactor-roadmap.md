# DataAgent Analyst Refactor Roadmap

## Refactor Goal

Transform DataAgent Analyst from a feature demonstration into a complete AI Data Analyst Workbench.

## R1: Analysis Foundation v2

- Better semantic role detection
- Canonical analysis context
- Data quality dimensions
- Target analysis
- Actionable findings
- Recommended next actions

## R2: Visualization Lab v2

- Apache ECharts
- Missing value heatmap
- Correlation heatmap
- Numeric histogram
- Boxplot
- Target distribution
- Feature versus target analysis
- Chart builder

## R3: ML Workbench v2

- Task detection
- Feature selection
- Datetime feature extraction
- Text vectorization
- Cross-validation
- Classification
- Regression
- Clustering
- Anomaly detection
- Expanded model options

## R4: Model Explainability

- ROC and precision-recall curves
- Residual analysis
- Error samples
- Permutation importance
- SHAP
- LLM model interpretation

## R5: Structured LLM Analyst

- Canonical analysis context input
- Validated JSON output
- Executive summary
- Key findings
- Data quality risks
- Model interpretation
- Business recommendations
- Next experiments
- Limitations

## R6: Dataset Knowledge RAG

- Dataset knowledge chunks
- Qdrant payload metadata
- Dataset-scoped retrieval
- Retrieved source cards
- Answers with citations
- Suggested questions

## R7: True Agent Workflow

- Dynamic planner
- Tool registry
- EDA Analyst
- Visualization Strategist
- ML Trainer
- Model Evaluator
- RAG Retriever
- LLM Explainer
- Report Writer
- Tool call events and warnings

## R8: AI Data Control Center UI

- Fixed top navigation
- Left sidebar
- Right AI insight panel
- Dataset selector
- Infrastructure status
- Cytoscape agent graph
- ECharts dashboard
- GSAP workflow animation
- Improved empty, loading, and error states
